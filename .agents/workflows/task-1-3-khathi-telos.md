# Workflow: Task 1.3 — Nghiên cứu khả thi (TELOS)

- Chạy `/mo-phien` trước (nạp ngữ cảnh, xác định vai, chờ tôi duyệt).
- Đóng vai ANALYST.

### 2. Phân tích 5 khía cạnh (mỗi khía cạnh: ĐẠT/KHÔNG + căn cứ)

### 3. Tổng hợp
- Bảng 5 dòng + kết luận KHẢ THI/CẦN ĐIỀU CHỈNH. Khía cạnh nào "Không đạt" → đề xuất thu hẹp phạm vi.
