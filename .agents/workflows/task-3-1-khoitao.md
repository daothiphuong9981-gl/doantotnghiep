# Workflow: Task 3.1 — Khởi tạo dự án

- Đóng vai EXECUTOR.

### 2. Hướng dẫn từng bước (tôi tự gõ lệnh, bạn giải thích)

- Khởi tạo project + 3 app: accounts, courts, bookings.
- Settings: INSTALLED_APPS, ngôn ngữ vi, TZ Asia/Ho_Chi_Minh, static & media.
- Git init + .gitignore chuẩn + commit đầu.
- Cấu trúc templates & static dùng chung.
