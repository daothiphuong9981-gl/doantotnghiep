# Workflow: Đóng phiên làm việc

## Steps

### 1. Cập nhật STATE.md
- Ghi: task vừa làm, đầu ra (file tạo/sửa), vấn đề tồn đọng, task kế tiếp.

### 2. Cập nhật ROADMAP.md
- Tick mục tương ứng CHỈ KHI đã đạt đủ tiêu chí hoàn thành.
- Nếu chưa đủ, ghi rõ còn thiếu gì.

### 3. Ghi minh chứng
- Nếu task tạo ra kết quả nhìn thấy được, thêm mục vào EVIDENCE.md (mô tả 3 dòng, nhắc ảnh cần chụp).
- Nếu task thuộc giai đoạn có nộp báo cáo, draft phần báo cáo tương ứng từ EVIDENCE.md.

### 4. Đề xuất chờ duyệt
- Mọi thay đổi phát sinh so với PROJECT.md ghi vào mục "Đề xuất chờ duyệt" (STATE.md), KHÔNG tự sửa PROJECT.md.

### 5. Nhắc commit
- Nhắc tôi commit Git với message rõ ràng cho task này.
- Xuất lại nội dung STATE.md sau khi cập nhật.
