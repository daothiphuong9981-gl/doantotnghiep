# Workflow: Giám sát luồng (cuối mỗi tuần)
- Đóng vai SUPERVISOR. Đầu vào: PROJECT.md (FR) + ROADMAP.md + mã nguồn hiện tại.

## Steps
### 1. Đối chiếu
- Bảng 3 cột: FR – Trạng thái (Xong/Đang/Chưa) – Ghi chú lệch.

### 2. Cảnh báo
- (a) FR Must chưa làm mà lẽ ra tuần này xong.
- (b) Thứ đã code nhưng KHÔNG thuộc FR nào (scope creep).
- (c) Code lệch thiết kế đã duyệt.

### 3. Kết luận
- Dự án có ĐÚNG LUỒNG không, cần chấn chỉnh gì trước tuần sau.
